-target type (output)
Specify target file type (vcd, svcd, dvd, dv, dv50). type may be prefixed
with pal-, ntsc- or film- to use the corresponding standard. All the format options (bitrate,
codecs, buffer sizes) are then set automatically. You can just type:
ffmpeg -i myfile.avi -target vcd /tmp/vcd.mpg
Nevertheless you can specify additional options as long as you know they do not conflict with the
standard, as in:
ffmpeg -i myfile.avi -target vcd -bf 2 /tmp/vcd.mpg
To set for NTSC DVD File it would be
ffmpeg -i myfile.avi -traget ntsc-dvd /home/brettsalemink/Videos/mytranscodefile.mpg
Example of encoding to MPEG-4 with a selective profile:
ffmpeg -i input.mp4 -c:v libx264 -profile:v high -level:v 4.0 -c:a copy output.mp4
