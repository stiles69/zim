DVDA-Author-Help


Content-Type: text/x-zim-wiki Wiki-Format: zim 0.4




dvda-author 2016-dev creates high-resolution DVD-Audio discs



from .wav, .flac and other audio files.



Usage: dvda-author [OPTION]...



Options:



Output options



-h, --help Diplay this help.



-v, --version Diplay version.



-q, --quiet Quiet mode.



-d, --debug Increased verbosity (debugging level)



-t, --veryverbose Like -d with enhanced verbosity for sample counts.



, --maxverbose Like -t with maximum verbosity on audio buffers (devel info).




--no-output Does not produce any file structure except for --fixwav.




<div style='padding-left: 180pt'>

Computations will be performed.







-P, --pause Insert a final pause before exiting.



-P0, --pause=0 Suppress a final pause before exiting



<div style='padding-left: 180pt'>
if specified in configuration file.




-l, --log path Ouput a log to filepath.



<div style='padding-left: 180pt'>
Argument must be supplied.




-L, --logrefresh path Same as -l but prior log will be erased on launching again.




--loghtml Transform the log into HTML format, with colorized messages, adjacent to log with extension .html
added.




-k, --text Generates text table in IFO files



<div style='padding-left: 180pt'>
Under development, implemented for 1-group discs.




<div style='padding-left: 180pt'>
Use file information as arguments separated by commas.




-W, --disable-lexer Deactivates configuration file parsing.



Playback options



-a, --autoplay Launches playback on loading disc.



Authoring options



Soundfile authoring






Supported audio types: .wav





<div style='padding-left: 180pt'>
.flac and .oga (Ogg FLAC, see below)






<div style='padding-left: 180pt'>
SoX-supported formats with -S enabled






<div style='padding-left: 180pt'>
except for lossy formats.


--project [file] Launches a dvda-author project.




<div style='padding-left: 180pt'>
The project file should have the same syntax as dvda-author.conf [see file HOW_TO.conf]




<div style='padding-left: 180pt'>
By default, launches dvda-author.dap in the current directory.




-i, --input directory Input directory with audio files.



<div style='padding-left: 180pt'>
Each subdirectory is a group.




-o, --output directory Output directory.



-x, --extract {disc or directory}[1:1,...,t1n-2:1,...,t2n-...-N:1,...,tNn] Extract
DVD-Audio to directory -o.



<div style='padding-left: 180pt'>
Groups are labelled g1, g2 in output directory.




<div style='padding-left: 180pt'>
Optional hyphen-separated list of groups to be extracted






<div style='padding-left: 180pt'>
may be appended to disc path.






<div style='padding-left: 180pt'>
Groups ranks are followed by colons.




<div style='padding-left: 180pt'>
Tracks to be extracted may be listed separated by commas after colon.





--aob-extract AOB[,AOB...] Direct AOB file audio content extraction.




<div style='padding-left: 180pt'>
Unlike -x, this option does not use .IFO files. Use this option in
combination with -o. No wav header is generated.




<div style='padding-left: 180pt'>
Several AOB files may be listed, separated by commas.





--aob2wav AOB[,AOB...] Like --aob-extract but a wav header is prepended
to audio content.





--log-decode [AOB] Decode AOB file and log MPEG specifics. Should be
used only in conjunction with --outfile





--outfile [file] Path to the log generated by --log-decode. Caution :
should be alone in its own directory.




-p, --startsector NNN Specify the number of the first sector



<div style='padding-left: 180pt'>
of the AUDIO_PP.IFO file in the output of mkisofs.




<div style='padding-left: 180pt'>
If NNN=0, falling back on 281 (default).






<div style='padding-left: 180pt'>
Without -p start sector will be computed automatically.




-g You may specify up to 9 groups of tracks.



<div style='padding-left: 180pt'>
Minimum: one group.
Enter full path to files if input directory is not set




<div style='padding-left: 180pt'>
by [-i].




-z, BROKEN. Separate two consecutive titles when files have
same audio



<div style='padding-left: 180pt'>
characteritics within a group.(

-Z, --playlist You may specify up to 9 group copies.



<div style='padding-left: 180pt'>
Total number of groups and copy groups should not
exceed 9.

-n, --no-videozone Do not generate an empty VIDEO_TS
directory.



-w, --rights Access rights to directories created
(octal values)



-c, --cga Enter channel group assignment right after
group (-g).



-F, --fixwav(options) Bad wav headers will be fixed by
fixwav.



<div style='padding-left: 180pt'>
Can be run alone without DVD-AUDIO output by adding
--nooutput.




-f, --fixwav-virtual(options) Use .wav header repair
utility



<div style='padding-left: 180pt'>
without any write operation.




-S, --sox Use SoX to convert files to .wav.



<div style='padding-left: 180pt'>
By default, only flac, Ogg FLAC




<div style='padding-left: 180pt'>
and .wav files are accepted.





--padding Reverse default behaviour for
transition between audio tracks with
identical




<div style='padding-left: 180pt'>
characteristics (number of channels, bit
depth and sample rate).




<div style='padding-left: 180pt'>
If necessary, audio will be padded with
0s instead of being joined (default).




<div style='padding-left: 180pt'>
Use --pad-cont for padding with
last-known byte.




-C, --pad-cont When padding, pad with
last known byte, not 0. See --padding
above.



<div style='padding-left: 180pt'>
Deactivates --lossy-rounding




-L, --lossy-rounding Sample count
rounding will be performed by cutting
audio files



<div style='padding-left: 180pt'>
instead of padding (see --padding
and --pad-cont).




<div style='padding-left: 180pt'>
Deactivates --pad-cont and
--padding.
