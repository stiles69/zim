Youtube-dl Manual


Created Wednesday 04 July 2018



youtube-dl [OPTIONS] URL [URL...]


OPTIONS


-h, --help Print this help text and exit
--version Print program version and exit
-U, --update Update this program to latest version. Make
<div style='padding-left: 240pt'>
	sure that you have sufficient permissions
	(run with sudo if needed)

	-i, --ignore-errors Continue on download errors, for example to
	<div style='padding-left: 240pt'>
		skip unavailable videos in a playlist

		--abort-on-error Abort downloading of further videos (in the
		<div style='padding-left: 240pt'>
			playlist or the command line) if an error
			occurs

			--dump-user-agent Display the current browser identification
			--list-extractors List all supported extractors
			--extractor-descriptions Output descriptions of all supported
			<div style='padding-left: 240pt'>
				extractors

				--force-generic-extractor Force extraction to use the generic
				<div style='padding-left: 240pt'>
					extractor

					--default-search PREFIX Use this prefix for unqualified URLs. For
					<div style='padding-left: 240pt'>
						example "gvsearch2:" downloads two videos
						from google videos for youtube-dl "large
						apple". Use the value "auto" to let
						youtube-dl guess ("auto_warning" to emit a
						warning when guessing). "error" just throws
						an error. The default value "fixup_error"
						repairs broken URLs, but emits an error if
						this is not possible instead of searching.

						--ignore-config Do not read configuration files. When given
						<div style='padding-left: 240pt'>
							in the global configuration file
							/etc/youtube-dl.conf: Do not read the user
							configuration in ~/.config/youtube-
							dl/config (%APPDATA%/youtube-dl/config.txt
							on Windows)

							--config-location PATH Location of the configuration file; either
							<div style='padding-left: 240pt'>
								the path to the config or its containing
								directory.

								--flat-playlist Do not extract the videos of a playlist,
								<div style='padding-left: 240pt'>
									only list them.

									--mark-watched Mark videos watched (YouTube only)
									--no-mark-watched Do not mark videos watched (YouTube only)
									--no-color Do not emit color codes in output
									Network Options:
									--proxy URL Use the specified HTTP/HTTPS/SOCKS proxy.
									<div style='padding-left: 240pt'>
										To enable SOCKS proxy, specify a proper
										scheme. For example

										string (--proxy "") for direct connection

										--socket-timeout SECONDS Time to wait before giving up, in seconds
										--source-address IP Client-side IP address to bind to
										-4, --force-ipv4 Make all connections via IPv4
										-6, --force-ipv6 Make all connections via IPv6
										Geo Restriction:
										--geo-verification-proxy URL Use this proxy to verify the IP address for
										<div style='padding-left: 240pt'>
											some geo-restricted sites. The default
											proxy specified by --proxy (or none, if the
											option is not present) is used for the
											actual downloading.

											--geo-bypass Bypass geographic restriction via faking
											<div style='padding-left: 240pt'>
												X-Forwarded-For HTTP header

												--no-geo-bypass Do not bypass geographic restriction via
												<div style='padding-left: 240pt'>
													faking X-Forwarded-For HTTP header

													--geo-bypass-country CODE Force bypass geographic restriction with
													<div style='padding-left: 240pt'>
														explicitly provided two-letter ISO 3166-2
														country code

														--geo-bypass-ip-block IP_BLOCK Force bypass geographic restriction with
														<div style='padding-left: 240pt'>
															explicitly provided IP block in CIDR
															notation

															Video Selection:
															--playlist-start NUMBER Playlist video to start at (default is 1)
															--playlist-end NUMBER Playlist video to end at (default is last)
															--playlist-items ITEM_SPEC Playlist video items to download. Specify
															<div style='padding-left: 240pt'>
																indices of the videos in the playlist
																separated by commas like: "--playlist-items
																1,2,5,8" if you want to download videos
																indexed 1, 2, 5, 8 in the playlist. You can
																specify range: "--playlist-items
																1-3,7,10-13", it will download the videos
																at index 1, 2, 3, 7, 10, 11, 12 and 13.

																--match-title REGEX Download only matching titles (regex or
																<div style='padding-left: 240pt'>
																	caseless sub-string)

																	--reject-title REGEX Skip download for matching titles (regex or
																	<div style='padding-left: 240pt'>
																		caseless sub-string)

																		--max-downloads NUMBER Abort after downloading NUMBER files
																		--min-filesize SIZE Do not download any videos smaller than
																		<div style='padding-left: 240pt'>
																			SIZE (e.g. 50k or 44.6m)

																			--max-filesize SIZE Do not download any videos larger than SIZE
																			<div style='padding-left: 240pt'>
																				(e.g. 50k or 44.6m)

																				--date DATE Download only videos uploaded in this date
																				--datebefore DATE Download only videos uploaded on or before
																				<div style='padding-left: 240pt'>
																					this date (i.e. inclusive)

																					--dateafter DATE Download only videos uploaded on or after
																					<div style='padding-left: 240pt'>
																						this date (i.e. inclusive)

																						--min-views COUNT Do not download any videos with less than
																						<div style='padding-left: 240pt'>
																							COUNT views

																							--max-views COUNT Do not download any videos with more than
																							<div style='padding-left: 240pt'>
																								COUNT views

																								--match-filter FILTER Generic video filter. Specify any key (see
																								<div style='padding-left: 240pt'>
																									the "OUTPUT TEMPLATE" for a list of
																									available keys) to match if the key is
																									present, !key to check if the key is not
																									present, key > NUMBER (like "comment_count
																									> 12", also works with >=, <, <=, !=, =) to
																									compare against a number, key = 'LITERAL'
																									(like "uploader = 'Mike Smith'", also works
																									with !=) to match against a string literal
																									and &amp; to require multiple matches. Values
																									which are not known are excluded unless you
																									put a question mark (?) after the operator.
																									For example, to only match videos that have
																									been liked more than 100 times and disliked
																									less than 50 times (or the dislike
																									functionality is not available at the given
																									service), but who also have a description,
																									use --match-filter "like_count > 100 &amp;
																									dislike_count <? 50 &amp; description" .

																									--no-playlist Download only the video, if the URL refers
																									<div style='padding-left: 240pt'>
																										to a video and a playlist.

																										--yes-playlist Download the playlist, if the URL refers to
																										<div style='padding-left: 240pt'>
																											a video and a playlist.

																											--age-limit YEARS Download only videos suitable for the given
																											<div style='padding-left: 240pt'>
																												age

																												--download-archive FILE Download only videos not listed in the
																												<div style='padding-left: 240pt'>
																													archive file. Record the IDs of all
																													downloaded videos in it.

																													--include-ads Download advertisements as well
																													<div style='padding-left: 240pt'>
																														(experimental)



																														Download Options:


																														-r, --limit-rate RATE Maximum download rate in bytes per second
																														<div style='padding-left: 240pt'>
																															(e.g. 50K or 4.2M)

																															-R, --retries RETRIES Number of retries (default is 10), or
																															<div style='padding-left: 240pt'>
																																"infinite".

																																--fragment-retries RETRIES Number of retries for a fragment (default
																																<div style='padding-left: 240pt'>
																																	is 10), or "infinite" (DASH, hlsnative and
																																	ISM)

																																	--skip-unavailable-fragments Skip unavailable fragments (DASH, hlsnative
																																	<div style='padding-left: 240pt'>
																																		and ISM)

																																		--abort-on-unavailable-fragment Abort downloading when some fragment is not
																																		<div style='padding-left: 240pt'>
																																			available

																																			--keep-fragments Keep downloaded fragments on disk after
																																			<div style='padding-left: 240pt'>
																																				downloading is finished; fragments are
																																				erased by default

																																				--buffer-size SIZE Size of download buffer (e.g. 1024 or 16K)
																																				<div style='padding-left: 240pt'>
																																					(default is 1024)

																																					--no-resize-buffer Do not automatically adjust the buffer
																																					<div style='padding-left: 240pt'>
																																						size. By default, the buffer size is
																																						automatically resized from an initial value
																																						of SIZE.

																																						--http-chunk-size SIZE Size of a chunk for chunk-based HTTP
																																						<div style='padding-left: 240pt'>
																																							downloading (e.g. 10485760 or 10M) (default
																																							is disabled). May be useful for bypassing
																																							bandwidth throttling imposed by a webserver
																																							(experimental)

																																							--playlist-reverse Download playlist videos in reverse order
																																							--playlist-random Download playlist videos in random order
																																							--xattr-set-filesize Set file xattribute ytdl.filesize with
																																							<div style='padding-left: 240pt'>
																																								expected file size

																																								--hls-prefer-native Use the native HLS downloader instead of
																																								<div style='padding-left: 240pt'>
																																									ffmpeg

																																									--hls-prefer-ffmpeg Use ffmpeg instead of the native HLS
																																									<div style='padding-left: 240pt'>
																																										downloader

																																										--hls-use-mpegts Use the mpegts container for HLS videos,
																																										<div style='padding-left: 240pt'>
																																											allowing to play the video while
																																											downloading (some players may not be able
																																											to play it)

																																											--external-downloader COMMAND Use the specified external downloader.
																																											<div style='padding-left: 240pt'>
																																												Currently supports
																																												aria2c,avconv,axel,curl,ffmpeg,httpie,wget

																																												--external-downloader-args ARGS Give these arguments to the external
																																												<div style='padding-left: 240pt'>
																																													downloader



																																													Filesystem Options:


																																													-a, --batch-file FILE File containing URLs to download ('-' for
																																													<div style='padding-left: 240pt'>
																																														stdin), one URL per line. Lines starting
																																														with '#', ';' or ']' are considered as
																																														comments and ignored.

																																														--id Use only video ID in file name
																																														-o, --output TEMPLATE Output filename template, see the "OUTPUT
																																														<div style='padding-left: 240pt'>
																																															TEMPLATE" for all the info

																																															--autonumber-start NUMBER Specify the start value for %(autonumber)s
																																															<div style='padding-left: 240pt'>
																																																(default is 1)

																																																--restrict-filenames Restrict filenames to only ASCII
																																																<div style='padding-left: 240pt'>
																																																	characters, and avoid "&amp;" and spaces in
																																																	filenames

																																																	-w, --no-overwrites Do not overwrite files
																																																	-c, --continue Force resume of partially downloaded files.
																																																	<div style='padding-left: 240pt'>
																																																		By default, youtube-dl will resume
																																																		downloads if possible.

																																																		--no-continue Do not resume partially downloaded files
																																																		<div style='padding-left: 240pt'>
																																																			(restart from beginning)

																																																			--no-part Do not use .part files - write directly
																																																			<div style='padding-left: 240pt'>
																																																				into output file

																																																				--no-mtime Do not use the Last-modified header to set
																																																				<div style='padding-left: 240pt'>
																																																					the file modification time

																																																					--write-description Write video description to a .description
																																																					<div style='padding-left: 240pt'>
																																																						file

																																																						--write-info-json Write video metadata to a .info.json file
																																																						--write-annotations Write video annotations to a
																																																						<div style='padding-left: 240pt'>
																																																							.annotations.xml file

																																																							--load-info-json FILE JSON file containing the video information
																																																							<div style='padding-left: 240pt'>
																																																								(created with the "--write-info-json"
																																																								option)

																																																								--cookies FILE File to read cookies from and dump cookie
																																																								<div style='padding-left: 240pt'>
																																																									jar in

																																																									--cache-dir DIR Location in the filesystem where youtube-dl
																																																									<div style='padding-left: 240pt'>
																																																										can store some downloaded information
																																																										permanently. By default
																																																										$XDG_CACHE_HOME/youtube-dl or
																																																										~/.cache/youtube-dl . At the moment, only
																																																										YouTube player files (for videos with
																																																										obfuscated signatures) are cached, but that
																																																										may change.

																																																										--no-cache-dir Disable filesystem caching
																																																										--rm-cache-dir Delete all filesystem cache files
																																																										Thumbnail images:
																																																										--write-thumbnail Write thumbnail image to disk
																																																										--write-all-thumbnails Write all thumbnail image formats to
																																																										disk
																																																										--list-thumbnails Simulate and list all available thumbnail
																																																										<div style='padding-left: 240pt'>
																																																											formats

																																																											Verbosity / Simulation Options:
																																																											-q, --quiet Activate quiet mode
																																																											--no-warnings Ignore warnings
																																																											-s, --simulate Do not download the video and do not write
																																																											<div style='padding-left: 240pt'>
																																																												anything to disk

																																																												--skip-download Do not download the video
																																																												-g, --get-url Simulate, quiet but print URL
																																																												-e, --get-title Simulate, quiet but print title
																																																												--get-id Simulate, quiet but print id
																																																												--get-thumbnail Simulate, quiet but print thumbnail URL
																																																												--get-description Simulate, quiet but print video
																																																												description
																																																												--get-duration Simulate, quiet but print video length
																																																												--get-filename Simulate, quiet but print output filename
																																																												--get-format Simulate, quiet but print output format
																																																												-j, --dump-json Simulate, quiet but print JSON information.
																																																												<div style='padding-left: 240pt'>
																																																													See the "OUTPUT TEMPLATE" for a description
																																																													of available keys.

																																																													-J, --dump-single-json Simulate, quiet but print JSON
																																																													information
																																																													<div style='padding-left: 240pt'>
																																																														for each command-line argument. If the URL
																																																														refers to a playlist, dump the whole
																																																														playlist information in a single line.

																																																														--print-json Be quiet and print the video information as
																																																														<div style='padding-left: 240pt'>
																																																															JSON (video is still being downloaded).

																																																															--newline Output progress bar as new lines
																																																															--no-progress Do not print progress bar
																																																															--console-title Display progress in console titlebar
																																																															-v, --verbose Print various debugging information
																																																															--dump-pages Print downloaded pages encoded using base64
																																																															<div style='padding-left: 240pt'>
																																																																to debug problems (very verbose)

																																																																--write-pages Write downloaded intermediary pages to
																																																																<div style='padding-left: 240pt'>
																																																																	files in the current directory to debug
																																																																	problems

																																																																	--print-traffic Display sent and read HTTP traffic
																																																																	-C, --call-home Contact the youtube-dl server for
																																																																	debugging
																																																																	--no-call-home Do NOT contact the youtube-dl server
																																																																	for
																																																																	<div style='padding-left: 240pt'>
																																																																		debugging

																																																																		Workarounds:
																																																																		--encoding ENCODING Force the specified encoding
																																																																		(experimental)
																																																																		--no-check-certificate Suppress HTTPS certificate
																																																																		validation
																																																																		--prefer-insecure Use an unencrypted connection to
																																																																		retrieve
																																																																		<div style='padding-left: 240pt'>
																																																																			information about the video. (Currently
																																																																			supported only for YouTube)

																																																																			--user-agent UA Specify a custom user agent
																																																																			--referer URL Specify a custom referer, use if the
																																																																			video
																																																																			<div style='padding-left: 240pt'>
																																																																				access is restricted to one domain

																																																																				--add-header FIELD:VALUE Specify a custom HTTP
																																																																				header and its value,
																																																																				<div style='padding-left: 240pt'>
																																																																					separated by a colon ':'. You can use this
																																																																					option multiple times

																																																																					--bidi-workaround Work around terminals that lack
																																																																					<div style='padding-left: 240pt'>
																																																																						bidirectional text support. Requires bidiv
																																																																						or fribidi executable in PATH

																																																																						--sleep-interval SECONDS Number of seconds to
																																																																						sleep before each
																																																																						<div style='padding-left: 240pt'>
																																																																							download when used alone or a lower bound
																																																																							of a range for randomized sleep before each
																																																																							download (minimum possible number of
																																																																							seconds to sleep) when used along with
																																																																							--max-sleep-interval.

																																																																							--max-sleep-interval SECONDS Upper bound of a
																																																																							range for randomized sleep
																																																																							<div style='padding-left: 240pt'>
																																																																								before each download (maximum possible
																																																																								number of seconds to sleep). Must only be
																																																																								used along with --min-sleep-interval.



																																																																								Video Format Options:


																																																																								-f, --format FORMAT Video format code, see the
																																																																								"FORMAT
																																																																								<div style='padding-left: 240pt'>
																																																																									SELECTION" for all the info

																																																																									<div style='padding-left: 240pt'>

																																																																													--write-auto-sub Write automatically

																																																																																			<div style='padding-left: 240pt'>

																																																																																																				May not be

																																																																																																				worstvideo: Select

																																																																																																				May not be

																																																																																																				May not be
																																																																																																				available.
																																																																																																				worstaudio: Select
																																																																																																				the worst quality
																																																																																																				audio only-format.
																																																																																																				May not be
																																																																																																				available.
																																																																																																				For example, to
																																																																																																				download the worst
																																																																																																				quality video-only
																																																																																																				format you can use
																																																																																																				-f worstvideo.



																																																																																																				If you want to
																																																																																																				download multiple
																																																																																																				videos and they
																																																																																																				don't have the same

																																																																																																				download format 18
																																																																																																				if it's available,
																																																																																																				otherwise it will
																																																																																																				complain that no
																																																																																																				suitable formats
																																																																																																				are available for

																																																																																																				If you want to
																																																																																																				download several
																																																																																																				formats of the same

																																																																																																				formats, of course
																																																																																																				if they are

																																																																																																				example combined
																																																																																																				with the precedence
																																																																																																				feature: -f

																																																																																																				You can also filter

																																																																																																				brackets, as in -f

																																																																																																				(or -f
																																																																																																				"[filesize>10M]").



																																																																																																				The following
																																																																																																				numeric meta fields
																																																																																																				can be used with
																																																																																																				comparisons <,
																																																																																																				<=, >, >=,
																																																																																																				= (equals), != (not
																																																																																																				equals):



																																																																																																				filesize: The
																																																																																																				number of bytes, if
																																																																																																				known in advance
																																																																																																				width: Width of the
																																																																																																				video, if known
																																																																																																				height: Height of
																																																																																																				the video, if known

																																																																																																				and video in KBit/s

																																																																																																				bitrate in KBit/s
																																																																																																				vbr: Average video
																																																																																																				bitrate in KBit/s
																																																																																																				asr: Audio sampling
																																																																																																				rate in Hertz
																																																																																																				fps: Frame rate
																																																																																																				Also filtering work
																																																																																																				for comparisons =
																																																																																																				(equals), != (not
																																																																																																				equals), ^= (begins
																																																																																																				with), $= (ends
																																																																																																				with), *=

																																																																																																				following string

																																																																																																				ext: File extension

																																																																																																				audio codec in use
																																																																																																				vcodec: Name of the
																																																																																																				video codec in use
																																																																																																				container: Name of
																																																																																																				the container
																																																																																																				format

																																																																																																				description of the
																																																																																																				format
																																																																																																				Note that none of
																																																																																																				the aforementioned
																																																																																																				meta fields are
																																																																																																				guaranteed to be
																																																																																																				present since this
																																																																																																				solely depends on
																																																																																																				the metadata
																																																																																																				obtained by
																																																																																																				particular
																																																																																																				extractor, i.e. the

																																																																																																				the value is not

																																																																																																				Videos can be
																																																																																																				filtered by their
																																																																																																				upload date using

																																																																																																				formats:



																																																																																																				Absolute dates:
																																																																																																				Dates in the format
																																																																																																				YYYYMMDD.
																																																																																																				Relative dates:
																																																																																																				Dates in the format
																																																																																																				(now|today)[+-][0-9](day|week|month|year)(s)?
																																																																																																				Examples:



																																																																																																				# Download only the
																																																																																																				videos uploaded in
																																																																																																				the last 6 months

																																																																																																				--dateafter
																																																																																																				now-6months



																																																																																																				# Download only the
																																																																																																				videos uploaded on
